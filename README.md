# Personal Logo – Initial Design

This repository contains my personal logo design based on my initial.  
The logo is intended for use as part of my personal brand and software development portfolio.

## Overview
- **Concept:** Initial-based logo
- **Style:** Minimal, clean, modern
- **Primary Use:** Portfolio, GitHub profile, personal projects
- **Design Tool:** Figma

The goal of this design is to create a simple and scalable mark that works well across digital platforms, including GitHub, websites, and presentations.

## Design Process
The logo was designed in Figma, focusing on:
- Simplicity and clarity
- Readability at small sizes
- A neutral style that can adapt to different color themes
